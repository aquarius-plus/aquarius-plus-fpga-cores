## Memory map

| Memory address    | Description         |
| ----------------- | ------------------- |
| $000000 - $07FFFF | RAM (512kB)         |
| $800000 - $803FFF | VRAM (16kB)         |
| $810000 - $810FFF | Text RAM (4kB)      |
| $820000 - $8207FF | Character RAM (2kB) |
|                   | Sprite attributes   |
|                   | ESP UART            |
|                   | Video registers     |
|                   | Audio registers     |
|                   | Interrupt registers |

## Sprite attributes

- X [8:0]
- Y [7:0]
- Tile index [8:0]
- Attributes
