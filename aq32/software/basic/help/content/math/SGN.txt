SGN returns a value indicating the sign of a numeric expression:

    @  1 if the expression if positive
    @  0 if it is zero
    @ -1 if it is negative

*SGN(numeric-expression)*

    @ *numeric-expression*  Any numeric expression.

Example:
    PRINT SGN(1), SGN(-1), SNG(0)   'Output is:  1  -1  0
