WRITE Statement

Writes data to an open file.

WRITE [#]filenumber%, expressionlist

    @ *filenumber%*       The number of an open file.
    @ *expressionlist*    One or more variables or expressions, separated by
                        commas, whose values are written to the file.

    @ WRITE writes the items to the file in their binary form.
      Use PRINT # instead to write ASCII representations.
